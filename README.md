# Mental Health Text Analyzer

This project classifies text into one of the following mental health categories:

- Anxiety
- Depression
- Suicidal
- Normal

## Model Used
- TF-IDF Vectorizer
- Logistic Regression

## Dataset Files
Place these files inside the `data/` folder:

- mental_heath_unbalanced.csv
- mental_health_combined_test.csv
